How to play

Follow these steps to use the app from start to finish.

  1. Click Get started and enter student name and date.
  2. On the state page, select one or more states to practice.
  3. Check the number of questions shown for each state (minimum 20).
  4. Click Start practice to begin random questions from your selected states.
  5. Choose one answer option for each question.
  6. If your answer is wrong, the correct option is highlighted for review.
  7. Questions answered correctly are not repeated again.
  8. When all questions are answered correctly, the app automatically opens the transcript report.
  9. Use Download report (PDF) to save or share the transcript.

Login / identify yourself
